Prolog Arithmetic Operators
<p>
Prolog can be used to perform arithmetic. Prolog supports the common arithmetic operators:
<ul>
<li><code>+</code> addition</li>
<li><code>-</code> subtraction</li>
<li><code>*</code> multiplication</li>
<li><code>/</code> division</li>
<li><code>%</code> remainder</li>
</ul>
Arithmetic expressions can be used as the right-hand argument of the <code><a href="Is.html">is</a></code> operator 
to assign the result of the expression to a variable.
Arithmetic expressions will also be evaluated if used as arguments for the numeric comparison operators:
<code><a href="NumericEquality.html">=:=</a></code>, 
<code><a href="NumericInequality.html">=\=</a></code>, 
<code><a href="NumericLessThan.html">&lt;</a></code>, 
<code><a href="NumericLessThanOrEqual.html">=&lt;</a></code>, 
<code><a href="NumericGreaterThan.html">&gt;</a></code> and
<code><a href="NumericGreaterThanOrEqual.html">&gt;=</a></code>.
</p>
